# Competitive Landscape

## Category map

| Category | Examples | Gap FortMemory fills |
|----------|----------|----------------------|
| Agent memory APIs | Mem0, Zep, Letta | Crypto governance + human Markdown truth |
| Vector DBs | Chroma, Qdrant, sqlite-vec alone | Productized policy + receipts |
| Note apps + AI | Obsidian plugins, Copilot | Multi-agent API + FortSignal gate |
| Agentic auth | FortSignal, Mastercard VI, AP2 | Durable knowledge store |

## Head-to-head (honest)

| Product | Wins | Loses to FortMemory when… |
|---------|------|---------------------------|
| **Mem0** | Ubiquity, extraction UX, speed to “remembers you” | Customer needs local files + verifiable writes |
| **Zep / Graphiti** | Temporal knowledge graphs | Governance and simplicity matter more than graph |
| **Letta** | Agent-managed memory OS | User wants vault-first, not runtime lock-in |
| **DIY Markdown + RAG** | Private, simple | Needs enforcement, passports, audit |
| **FortSignal alone** | Best action auth | Needs a memory state plane |

## Differentiation pillars (only three)

1. **Verifiable memory operations** — FortSignal `signalId` on mutates  
2. **Human-readable source of truth** — Obsidian Markdown  
3. **Local-first, optional FortVault** — privacy default  

## Positioning line

> Mem0 makes agents remember. FortMemory makes agent memory **accountable**.

## What not to claim

- “SOTA memory benchmark” as launch narrative  
- “Replace Obsidian”  
- “Decentralized global brain”  

## GTM wedges

1. Local LLM communities (Ollama, LocalLLaMA)  
2. Existing FortSignal users (agent passports already live)  
3. Security/compliance teams fearing shadow AI memory  
4. Agent tool builders who already write files and need guardrails  

## Pricing posture vs competitors

- Free/open local core to earn trust (open-core)  
- Charge for FortVault team sync + support  
- Metering synergy with FortSignal verification volume  

## Moat thesis (realistic)

Moat is **not** embeddings. Moat is:

- FortSignal enforcement already shipping  
- Agent passports + NL policies + receipts  
- Willingness to stay local-first and boring on storage  
- Enterprise VI-aligned narrative applied beyond payments
